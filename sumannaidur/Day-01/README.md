# Day 01 - Java Basics

## Problem 1: Personal Info
- **Description**: Program to store and print personal information (Name, Age, Percentage, IsStudent).
- **File**: `PersonalInfo.java`

## Problem 2: Simple Calculator
- **Description**: Arithmetic operations (Add, Subtract, Multiply, Divide) using methods.
- **File**: `SimpleCalculator.java`

## Problem 3: Total Marks & Percentage
- **Description**: Calculates total marks and percentage for 5 subjects.
- **File**: `MarksCalculator.java`

## Problem 4: Temperature Converter
- **Description**: Converts Celsius to Fahrenheit.
- **File**: `TemperatureConverter.java`

## Problem 5: Salary Increment Program
- **Description**: Calculates final salary based on basic salary and bonus percentage.
- **File**: `SalaryCalculator.java`

## Problem 6: Swap Two Numbers
- **Description**: Swaps two numbers with and without using a third variable.
- **File**: `SwapNumbers.java`

## Problem 7: Factorial
- **Description**: Calculates the factorial of a number using a method.
- **File**: `Factorial.java`

## Problem 8: Static vs Instance Variable
- **Description**: Demonstrates the difference between static (class-level) and instance (object-level) variables.
- **File**: `VariableScopeDemo.java`
